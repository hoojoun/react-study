# 리엑트 스터디

```
src/
├── App.jsx
├── pages/
│   └──  managementPage.jsx # 관리 페이지의 최상단 컴포넌트, 상태관리
└── components/
    ├── common/               # 공통 컴포넌트
    │   └── button.jsx        # 버튼
    ├── sideBar               # 왼쪽 사이트바 컴포넌트
    │   └── index.jsx
    └── views/                # 오른쪽 뷰 컴포넌트
        ├── index.jsx         # 뷰 분기 컴포넌트(route)
        ├── addProjectView.jsx  # 프로젝트 추가 페이지
        ├── detailView.jsx    # 상세 프로젝트 페이지
        └── homeView.jsw      # 메인 페이지 및 에러 페이지

```

# 프로젝트 셋팅

npm install  
npm run dev
